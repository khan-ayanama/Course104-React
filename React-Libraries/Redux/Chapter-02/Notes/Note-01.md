# Middlewares

## Creating JSON server to serve the data

1. `Installation:`

    ```bash
        npm i --save-dev json-server
    ```

2. `Adding in package.json:`

   ```json
        "scripts": {
        "json-start":"json-server db.json",
        },
   ```

3. `Running server:`

    ```bash
        npm run json-start
    ```
